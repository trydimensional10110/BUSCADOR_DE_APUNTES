## DESCRPCION DEL PROYECTO

El programa "Buscador de apuntes" es una pagina para ordenadores y dispositivos móviles la cual busca que los alumnos de un mismo curso tenga su propia pagina web para subir lo que se hizo en el dia de la fecha en cierta materia para que no se pierdan los contenidos a lo largo del año. Esta estará dirigida para los alumnos de un solo curso, 6to 2da, y los admnisitradores.
